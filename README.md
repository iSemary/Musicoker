## Installation

#### React Configuration

```sh
    cd musicoker-frontend
```

```sh
    npm install
```

```sh
    npm start
```

#### Laravel Configuration

```sh
    cd musicoker-backend
```

```sh
    composer install
```

```sh
    php artisan key:generate
```

```sh
    php artisan migrate
```

```sh
    php artisan serve
```
